# L'Artisan Café Prototype

A premium sensory café storefront and tabletop order customizer built as a prototype web app.

This repository contains the source for a modern café experience with interactive menu browsing, order customization, staff tracking, and customer feedback.

## Features

- Interactive menu browsing and customizable orders
- Cart and checkout flow
- Rewards and loyalty-style UI
- Staff/admin portal with kitchen tracker
- Customer feedback and analytics-ready views

## Local setup

**Prerequisites:**
- Node.js 20+ (recommended)
- npm

1. Install dependencies:
   `npm install`
2. Copy the example environment file:
   `cp .env.example .env`
3. Run the development server:
   `npm run dev`

Open the app at `http://localhost:3000`.

## Scripts

- `npm run dev` — start the Vite development server
- `npm run build` — build the production assets
- `npm run preview` — preview the production build
- `npm run clean` — remove build artifacts
- `npm run lint` — type-check the project

## GitHub push instructions

1. Initialize a git repository:
   `git init`
2. Add files to version control:
   `git add .`
3. Commit your prototype:
   `git commit -m "Initial prototype commit for L'Artisan Café"`
4. Add your remote repository:
   `git remote add origin https://github.com/<your-username>/<repo-name>.git`
5. Push to GitHub:
   `git branch -M main && git push -u origin main`
